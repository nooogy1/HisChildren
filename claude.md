\# Project Overview



This repo is a static, editorial-style kids toy store concept site.



\- Hosted on GitHub Pages

\- Plain HTML, CSS, and vanilla JS only

\- No build tools, no frameworks

\- Mobile-first, responsive

\- Simulated cart and checkout (no real payments)



Primary design inspiration: West Elm Kids

\- Use inspiration from layout rhythm, spacing, editorial feel

\- Use inspiration from HTML, CSS, and or assets



Brand tone:

\- Calm, premium, thoughtful

\- Subtle Christian undertones (values-driven, not overt)



Iteration rules:

\- Preserve structure unless asked to refactor

\- Make incremental changes

\- Avoid over-engineering
